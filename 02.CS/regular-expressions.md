### 정규 표현식

```
\ : 문자 그대로 해석 
^ : 시작(부정을 나타내는 [^]과는 다르다)
$ : 끝
* : 0번 이상 반복되는 경우 매칭
+ : 1번 이상 반복되는 경우 매칭
? : 0 또는 1번 반복되는 경우 매칭
?? : ?와 같으나 더 짧은 것을 매칭
. : 개행 문자를 제외한 모든 단일 문자 매칭
(x) : 그룹화
  /ab+/g 
  ['abbb', 'abb', 'ab']
  /(ab)+/g 
  ['abab', 'ab']
(?:x) : 비 캡쳐링 그룹화
x(?=y) : 전방 탐색 y를 만족하면서 왼쪽에 x가 있는 경우
x(?!y) : 부정 전방 탐색 y를 만족하면서 왼쪽에 x가 없는 경우
(?<=y>)x : 후방 탐색
(?<!y>)x : 부정 후방 탐색
x|y : or /red|blue/g
{n} : n만큼 반복된 경우 매칭
{n,m} : n이상 m이하 반복된 경우 매칭
{n,}, {n,}? : n이상 반본된 경우 매칭
[xyz]
  const str = 'gray and grey'; 
  str.match(/gr[ae]y/g); //['gray', 'grey']
  /[A-Z]/;
  /[0-9]/;
  /[가-힇]/;
[^xyz]: 내부에 해당하지 않는 문자만 매칭
[\b]
\b : 다른 단어가 좌우에 없는 것만 매칭
\B : 다른 단어가 좌우에 있는 경우만 매칭
\cX
\d : [0-9]와 같음
\D : [^0-9]와 같음
\f
\n : 줄바꿈
\r
\s : 공백 매칭
\S : 공백이 아닌 것만 매칭
\t
\v
\w : [0-9a-zA-Z_]와 같음
\W : [^0-9a-zA-Z_]와 같음
\0
\n : n번째 그룹화와 대응한 역참조
  /(ab)\1/
  /(a)(b)\1\2/
\xhh
\uhhhh

```

```
(?<name>x), \k<name>
  캡쳐링에 이름을 정할 수 있다.
```

```
/abc/g
new Regex('abc', g)

g: 전역 검색
m: 다중 행 검색
i: 대소문자 구분 없이
s: .에 개행 문자도 매칭
u: 유니코드
y: 문자열의 현재 위치부터 검색 수행
```
